# Pinecone AI Club

Hands-on AI curriculum for developers with fullstack experience.

## Modules

| # | Module | Topics |
|---|--------|--------|
| 1 | **OpenAI API** | API requests, completions, temperature, shot prompting, chatbots |
| 2 | **Prompt Engineering** | Message roles, output formatting, CoT reasoning, business applications, chatbot development |

## Prerequisites

- 6+ months of fullstack development experience
- Basic TypeScript knowledge
- Node.js installed

## Setup

```bash
cp "1. OpenAI API/.env.example" "1. OpenAI API/.env"
# Add your OpenAI API key to .env

cd "1. OpenAI API"
npm install
```

## How It Works

Each module has numbered tasks (`task-XX-name.ts`) — complete them in order. Solutions are in the `solutions/` folder of each chapter.

Run a task:
```bash
npx ts-node task-01-api-request.ts
```
